# Mini LISP Interpreter

A interpreter of a prototype of LISP.

## Environment

- Windows 10
- WSL 2
    - Ubuntu 20.04 LTS
        - bison 2:3.5.1+dfsg-1
        - flex 2.6.4-6.2
        - gcc 4:9.3.0-1ubuntu2
- Visual Studio Code
    - Yash 0.2.6

## Implement Way

- C
- Yacc
- Lex

## Feature

- Type Checking
- Print Statements
    - Print Number
    - Print Boolean
- Numerical Operations
    - Addition
    - Subtraction
    - Multiplication
    - Division
    - Modulus
- Logical Operations
    - And
    - Or
    - Not
- Define Statements
    - Define Variable
    - Define Function
- Function
    - Anonymous Function
    - Named Function
- If-Else Statement
- Recursion
- Nested Function
- First-Class Function
- Closure
